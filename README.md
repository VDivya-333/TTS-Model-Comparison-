<h1 align="center">TTS Model Comparison 🗣️</h1>
<p align="center">Suno Bark vs. Facebook MMS</p>

This project provides a simple Python script to test and compare two popular Text-to-Speech (TTS) models:
- **Suno Bark** (`suno/bark-small`)
- **Facebook MMS** (`facebook/mms-tts-eng`)

The script automatically generates audio from a list of sentences, converts the audio back to text, and then scores how accurately each model performed.

## Features

- Compares `suno/bark-small` and `facebook/mms-tts-eng`.
- Uses `openai/whisper-tiny` for reliable ASR-based evaluation.
- Saves the generated audio (`.wav` files) for you to listen to.
- Creates a `tts_results.csv` file with the final accuracy scores for easy comparison.
- Uses a voice preset for the Suno Bark model to ensure higher-quality audio generation.

## ⚙️ How It Works

The core logic follows these steps for each sentence in the input file:

1.  **Generate Speech**: It takes a sentence and asks both the Suno and Facebook models to convert it to audio.
2.  **Save Audio**: It saves the audio from each model as a `.wav` file.
3.  **Transcribe Audio**: It listens to the generated audio using the Whisper (speech-to-text) model to see what words it hears.
4.  **Compare & Score**: It compares the transcribed text to the original sentence and calculates an accuracy score. A higher score means the generated audio was clearer and more accurate.

## 🚀 Getting Started

### Prerequisites

- **Python 3.8+**
- **PyTorch**: Install it from the official PyTorch website. Choose the right version for your system (CPU or GPU/CUDA).
- **FFmpeg**: This is a common dependency for audio processing. You may need to install it separately. You can download it from the official FFmpeg website.

### Installation

1.  **Clone the repository or download the files.**

2.  **Install the required Python packages:**

    It's a good practice to use a virtual environment.

    ```bash
    # Create and activate a virtual environment (optional but recommended)
    python -m venv venv
    source venv/bin/activate  # On Windows, use `venv\Scripts\activate`
    ```

    Create a file named `requirements.txt` with this content:
    ```
    transformers
    pandas
    numpy
    jiwer
    soundfile
    accelerate
    ```

    Then, install all the packages with one command:
    ```bash
    pip install -r requirements.txt
    ```

3.  **Create the input data file:**

    Create a file named `input.csv` in the same directory as the script. It must contain `id` and `text` columns.

    **Example `input.csv`:**
    ```csv
    id,text
    1,"Hello world, this is a test of the text to speech system."
    2,"The quick brown fox jumps over the lazy dog."
    3,"How much wood would a woodchuck chuck if a woodchuck could chuck wood?"
    ```

## ▶️ Usage

Simply run the Python script from your terminal:

```bash
python F_S_modelTest.py
```

The script will first download and cache the required models (this may take some time and disk space). It will then process each sentence from `input.csv`, printing the progress and accuracy scores to the console.

## Output

After the script finishes, you will find the following outputs:

- **`suno_outputaudio/`**: A folder containing the audio files generated by the Suno Bark model (e.g., `suno_audio1.wav`).
- **`facebook_outputaudio/`**: A folder containing the audio files generated by the Facebook MMS model (e.g., `facebook_audio1.wav`).
- **`tts_results.csv`**: A CSV file containing the final comparison, with columns for `id`, `text`, `fac_accuracy`, and `suno_accuracy`.
